# Nucleation tab

Appears when the simulation mode is set to **nucleation** on the [Structure options](structure-options.md) tab.

Configures the nucleation routine, which sweeps over a range of cluster sizes and supersaturations to produce nucleation statistics.

---

## Nucleation routine

| Field | Description |
|---|---|
| Start size | Smallest cluster size to simulate (number of molecules) |
| End size | Largest cluster size to simulate |
| Step size | Increment between successive cluster sizes |
| Number of simulations | How many independent simulations to run at each size/∆µ combination |
| Start ∆µ | Lowest supersaturation value to sweep (kcal/mol) |
| End ∆µ | Highest supersaturation value to sweep (kcal/mol) |
| Step ∆µ | Increment between supersaturation values |
