# Excess supersaturation tab

Appears when **Excess any** is set to **Yes** on the [Supersaturation profile](supersaturation.md) tab.

Defines additional supersaturation periods that are applied on top of the main profile — for example, to simulate a brief spike in driving force or a dilution step.

---

## Iterations section

| Field | Description |
|---|---|
| Number of excess entries | How many additional supersaturation periods to define; click **Generate fields** to create the corresponding rows |

## Per-period fields

One section is generated per excess period after clicking **Generate fields**.

| Field | Description |
|---|---|
| Tile supersaturation | Supersaturation value applied to tile-type interactions during this period |
| Grouped supersaturation | Supersaturation value applied to grouped molecules during this period |
| Ungrouped supersaturation | Supersaturation value applied to ungrouped molecules during this period |
