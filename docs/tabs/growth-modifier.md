# Growth modifier tab

Appears when the simulation mode is set to **growth_modifier** on the [Structure options](structure-options.md) tab.

Configures the poison/modifier sites that inhibit growth at specific crystal positions.

---

## Growth modifier settings

| Field | Description |
|---|---|
| Number of poison sites | How many distinct poison-site types to define; click **Generate fields** to create the corresponding rows |
| Poison attach | Probability that a modifier molecule attaches to a growth site |
| Poison remain | Probability that an attached modifier remains on a site during a growth step |

## Growth modifier sites

One row is generated per poison-site type after clicking **Generate fields**.

| Field | Description |
|---|---|
| Site (per type) | Identifier or coordinates of the poison-site type |
