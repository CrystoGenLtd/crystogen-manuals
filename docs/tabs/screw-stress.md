# Screw stress tab

Appears when the simulation mode is set to **screw_stress** on the [Structure options](structure-options.md) tab.

Defines the stress-field parameters around a screw dislocation.

---

## Screw stress section

| Field | Description |
|---|---|
| Radius stress | Radial extent of the stress field (Å) |
| Start stress | Stress value at the dislocation core |
| Length stress | Axial length of the stress field |
