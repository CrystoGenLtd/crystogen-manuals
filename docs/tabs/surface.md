# Surface tab

Appears when the simulation mode is set to **surface** on the [Structure options](structure-options.md) tab.

Configures growth on or near a substrate surface.

---

## Surface growth section

| Field | Description |
|---|---|
| Substrate | **Yes** / **No** — whether to include a substrate beneath the growing crystal |
| Thickness | Thickness of the substrate layer (Å) |
| Width | Lateral width of the simulation cell (Å) |
| h, k, l | Miller indices of the substrate surface plane |
| Energy | Interaction energy between the crystal and the substrate (kcal/mol) |
| Seed x / y / z | Initial position of the seed crystal relative to the substrate |
