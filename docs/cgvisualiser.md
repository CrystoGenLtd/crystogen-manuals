# CG Visualiser

!!! warning "Being phased out"
    CG Visualiser is being phased out. Its functionality is superseded by [CGAspects](https://github.com/CrystoGenLtd/cgaspects), which is cross-platform and actively maintained. New users should use CGAspects instead. CG Visualiser will continue to function for existing workflows but will not receive further updates.

CG Visualiser is a **Windows-only** application for visualising CrystoGen simulation output. It is included in the CrystoGen download for users who need it for existing workflows.

For macOS and Linux, [CGAspects](https://github.com/CrystoGenLtd/cgaspects) or [Ovito](https://ovito.org) are the recommended visualisation tools.

---

## Installation (Windows only)

CG Visualiser is included in the **CGVisualiser** folder of the CrystoGen download (register at [crystogen.org](https://crystogen.org) to receive the link).

Select the version matching your Windows installation:

| File | Windows version |
|---|---|
| `CG Visualiser v1.71_64bit.exe` | 64-bit (recommended) |
| `CG Visualiser v1.71_32bit.exe` | 32-bit |

Double-click the executable to run — no installation is required.

---

## Recommended alternative: CGAspects

[CGAspects](https://github.com/CrystoGenLtd/cgaspects) is the recommended replacement. It is available for Windows, macOS, and Linux, and is included in the CrystoGen download for all platforms. It supports Zingg diagrams, crystal shape rendering, and data processing for CrystoGen output.

The latest version of CGAspects can also be downloaded directly from its [GitHub releases page](https://github.com/CrystoGenLtd/cgaspects/releases).
